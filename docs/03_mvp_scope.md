# Alcance del MVP v1

## Dominio técnico inicial

Comparación y desarrollo de vigas de acero para claros simples.

## Caso guía inicial

Definir y desarrollar el elemento principal para salvar un claro simple entre columnas, con comparación explícita entre:

- viga laminada convencional
- viga armada de alma llena
- viga alveolar
- celosía

## Capacidades obligatorias del MVP

### Proyecto
- crear proyecto local
- definir normativa activa
- definir unidades e idioma
- asociar documentos autorizados

### Árbol
- crear nodo raíz de problema
- generar primer abanico de alternativas
- activar una rama principal
- conservar ramas descartadas
- reactivar una rama descartada
- clonar una rama existente

### Documentos
- registrar documentos fuente
- vincular nodos a referencias documentales
- abrir referencia desde nodo o cálculo

### Cálculo
- registrar hipótesis
- capturar variables con unidades
- ejecutar cálculos deterministas simples
- registrar sustituciones y resultados
- generar comprobaciones básicas

### Comparación
- comparar ramas por criterios homogéneos
- registrar justificación de la selección

### Salida
- generar reporte exportable básico
- conservar historial de cambios del árbol

## No entra en el MVP

- análisis estructural 3D completo
- integración BIM/CAD
- costos cuantificados de mercado
- planos automáticos
- optimización avanzada
- diseño sísmico completo de sistemas especiales
